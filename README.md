# ng-sonar-demo
Angular SonarTS/ TSLint Demo

This source code is part of a blogpost about running Javascript Unit Tests with Visual Studio Team Services using real browsers.

More info: [Improving Angular Style and code quality](https://yuriburger.net)

To setup the demo:

`npm install`

To build and run the TSLint demo: 

`npm run build`

`npm run lint`
